@sass-fairy/math

exp ()

Returns Euler’s number raised to the specified power.

Parameters​

$numbers
The power to which Euler’s number will be raise.
Type
Number

Return Value​

Number
The number representing Euler’s number raised to the given power.

Example​

@use '@sass-fairy/math';

@debug math.exp(2);
  // 7.3890560989

@debug math.exp(5);
  // 148.4131591026
